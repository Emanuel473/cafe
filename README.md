# ☕ Café com Você

Este é o meu primeiro projeto web, desenvolvido para praticar **HTML, CSS e JavaScript**. O site simula uma cafeteria onde o usuário pode escolher cafés e calcular o subtotal com acréscimos na página de pagamento.

## 🚀 Funcionalidades
- Galeria de Cafés com preços.
- Integração com JavaScript para cálculo de pedidos.
- Uso de `localStorage` para transição de dados entre páginas.
- Layout responsivo com CSS.

## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estrutura das páginas.
- **CSS3**: Estilização e design.
- **JavaScript**: Lógica de cálculo e interatividade.
- **Git/GitHub**: Controle de versão e hospedagem.

## 📂 Como visualizar
Você pode acessar o projeto rodando o arquivo `index.html` localmente ou através do link do GitHub Pages (se já tiver ativado).
